Shift

An automated upgrade tool for your Laravel projects.

How it works

Sign In

Sign in using your GitHub or Bitbucket account to allow Shift to quickly access your Laravel project.

Purchase a Shift

Purchase a Shift and specify which Laravel project you want to automatically upgrade.

Review Pull Request

Shift will submit a pull request for review so you can perform any additional changes and merge when you're ready.
